# Security Policy

QLite-OS is a freestanding operating system scaffold. Security issues are welcome if they affect build integrity, boot flow, memory safety, or repository hygiene.

## Reporting

If you find a security concern, describe:

- The affected file or area.
- The impact.
- Steps to reproduce, if available.
- Any suggested mitigation.

## Scope

Relevant topics include:

- Unsafe memory access in the kernel.
- Bootloader and entry-point correctness.
- Workflow or supply-chain problems.
- Accidental exposure of generated artifacts.

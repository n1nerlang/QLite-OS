# Icons

This directory holds icon assets for QLite-OS.

Current assets:

- `qlite-logo.svg` - the main QLite logo.
- `qlite-terminal.svg` - a terminal-themed icon for QLterm and shell surfaces.
- `qsearch.svg` - icon for QSearch.
- `qstudio.svg` - icon for QStudio.
- `qlpaint.svg` - icon for QLpaint.
- `qlcalc.svg` - icon for QLcalc.
- `qlfiles.svg` - icon for QLfiles.
- `qlnotes.svg` - icon for QLnotes.
- `qlmusic.svg` - icon for QLmusic.
- `qlsettings.svg` - icon for QLsettings.
- `qlfetch.svg` - icon for QLfetch.
- `qlterm.svg` - icon for QLterm.

Future additions can include per-app icons or exported image sizes for launchers and menus.

# Apps

QLite-OS includes a small set of built-in demo apps:

- `QSearch` for search-like queries.
- `QStudio` for an editor-style workspace.
- `QLpaint` for drawing and fill behavior.
- `QLcalc` for expression evaluation.
- `QLfiles` for tree browsing.
- `QLnotes` for note-taking.
- `QLmusic` for playback and queue display.
- `QLsettings` for system toggles.
- `QLfetch` for a Fastfetch-style system banner.
- `QLterm` for command execution and shell output.

Each app is currently rendered as a text UI demo.

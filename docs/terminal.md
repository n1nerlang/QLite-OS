# Terminal

The QLterm demo is built around a small command dispatcher.

Supported commands:

- `help`
- `stats pc`
- `ls /`, `ls /apps`, `ls /rust`
- `cat /motd`, `cat /readme`
- `echo ...`
- `fetch`
- `clear`

The `stats pc` command prints the logo and system stats block used by the Fastfetch-style view.
